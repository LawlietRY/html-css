Principais formatações

Negrito / Destaque

Existem duas maneiras de colocar negrito, uma forma com semantica, outra forma sem

Essa frase temos um termo em negrito Usando a tag B (não semantica)

Nesta frase por outro lado tenho um termo em destaque Usando a tag strong (forma semantica)

Italico / enfâse

Existem duas maneiras de colocar o termo em italico também, uma semantica e outra não

Essa frase terá o termo em forma italica usando a tag i (não semantica)

Nesta frase temos o termo em destaque usando a tag em (semantica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK

Texto Grande é pequeno

Estamos criando um <big> texto grande <big> (utilizando a tag big, apesar de estar obsoleta) e um texto pequeno (utilizando a tag small) nesse paragrafo

Texto Excluido

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado (usando a tag del)

Texto inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois (usando a tag ins) lembrando que ela é uma tag semantica

Também pode utilizar a tag u, apesar de ser obsoleta

Texto Sobreescrito

Utilizado para inserir coisas do tipo x20+3 utilizando a tag "sup"

Texto Subescrito

Para inserir coisas do tipo H2O utilizando a tag "sub"
